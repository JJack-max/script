## 🚗 #309（识别几乎不再访问的 S3 Bucket，运维开销最小）

**正确答案：✅ A（S3 Storage Lens）**

---

### 为什么是 A（官方/考试视角）

- **Amazon S3 Storage Lens**
  - **专门用于存储分析与成本优化**
  - 提供 **Bucket 级 / 账户级访问频率指标**
  - 可直接识别：
    - 很少访问
    - 长期未访问
    - 访问趋势下降的 Bucket
- **无需自建日志分析或查询系统**
- 是 AWS 官方推荐的 **低运维、统一视图** 存储分析方案

> 在考试语境中，“**识别很少或不再访问的 S3 Bucket**” 是 **Storage Lens 的典型适用场景**

---

### 为什么其他选项不对（考试扣分点）

- **B. S3 控制台仪表板**
  - 仅提供**基础、即时视图**
  - **不适合长期趋势分析**
  - 无法系统性识别“几乎不再访问”的 Bucket

- **C. CloudWatch + Athena**
  - 架构复杂
  - 明显 **运维开销高**
  - 违反题干“最少运维开销”

- **D. CloudTrail + CloudWatch Logs**
  - 面向 **审计 / 安全**
  - 成本与运维复杂度最高
  - 严重过度设计
