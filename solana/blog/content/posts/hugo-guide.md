# Hugo 入门指南

Hugo是一个用Go语言编写的快速静态网站生成器，非常适合构建博客、文档网站等。

## 为什么选择Hugo？

1. **速度快** - 生成速度非常快
2. **易于使用** - 简单的目录结构和配置
3. **功能丰富** - 支持多语言、分类、标签等
4. **灵活的模板** - 强大的模板系统

## 基本命令

```bash
hugo new site mysite      # 创建新网站
hugo server              # 启动本地服务器
hugo                     # 生成静态文件
```

## 项目结构

- `content/` - 站点内容
- `layouts/` - HTML模板
- `static/` - 静态文件
- `themes/` - Hugo主题
- `config.toml` - 站点配置

开始你的Hugo之旅吧！
