# Markdown 语法速查表

Markdown是一种轻量级的标记语言，易于学习和使用。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本格式

- **粗体** - `**粗体**`
- *斜体* - `*斜体*`
- ~~删除线~~ - `~~删除线~~`

## 列表

**无序列表：**
- 项目1
- 项目2
- 项目3

**有序列表：**
1. 第一项
2. 第二项
3. 第三项

## 代码

行内代码：`code`

代码块：
```python
def hello():
    print("Hello, World!")
```
